# Medical Dataset Cleaning Pipeline

This project demonstrates a basic workflow for cleaning and analyzing healthcare patient data using Python.

## Objectives
- Clean raw healthcare data (handling duplicates and missing values).
- Perform basic exploratory data analysis (EDA) on patient demographics and medical conditions.
- Visualize findings to identify disease trends.

## Files
- `healthcare_analysis.py`: The main script for processing and visualization.
- `cleaned_patient_data.csv`: The processed output ready for further research.
- `disease_distribution.png`: A bar chart visualizing the frequency of diseases.

## How to Run
1. Install dependencies: `pip install pandas matplotlib`
2. Run the script: `python healthcare_analysis.py`

## Technologies
- Python
- Pandas
- Matplotlib
